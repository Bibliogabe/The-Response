About

Cirrus representation of most recorded terms from interviews collected by group. Created by Gabriel Lopez with Voyant Tools. (https://voyant-tools.org)

The COVID-19 Pandemic was an unprecedented and tumultuous time for libraries across the world. Librarians had to adapt quickly as community shutdowns and social distancing policies were enacted in almost every facet of life. The collection that our group created demonstrates how librarians and other information professionals overcame the challenges created by the pandemic in order to serve patrons in their respective communities. This collection is dedicated to libraries of all types: Public, academic, school, and research. We have collected interviews, images, flyers, advertisements, and other electronic content with the hopes of showing how adaptable libraries can be in the face of such unknown circumstances. This collection is just a small insight into what we think libraries are capable of. The group chose a wide variety of objects, from multiple sources, in order to reflect the profound depth of changes to library spaces and services. Interviews from librarians of multiple types of libraries, copies of flyers promoting new digital venues for events, and photographs and video of how library spaces were alternated to facilitate social distancing. Our exhibit “The Response: Libraries of Texas” was chosen because the majority of projects collected prignitaed from Texas libraries. Group members chose objects that they felt exemplified the aim of the collection in relation to the variety of library types in Texas. 

Dublin Core was used for the collections metadata creation. The group consulted the CDP Dublin Core Metadata Best Practices Version 2.1 in order to formalize the data entry in the separate fields chosen to describe the objects. The group chose this metadata scheme in order to best describe the variety of objects collected. Geolocation was also recorded in order to demonstrate the multiple locations of libraries objects were collected from. Changes to the CDP were implemented in order to better reflect modern conventions and the collections overall aesthetic. For example, file size is referred to in kilobytes instead of the recommended bytes, common book title capitalization standards are used for object titles, and GETTY protocols were used for coverage information. The group also chose to utilize the Item Type information in Omeka. The fields chosen by the group are:

Title: object name, type of Library object originates from, and name of institution.
Subject: Using LCSH authority terms, each term in a separate file tags were used to supplement
Description: brief synopsis of item and item context
Creator: last name, first name
Date: YYYY-MM-DD format
Contributor: last name, first name where applicable
Rights: Creative Commons template displayed in html format, with additional information for items used with permission
Format: application and file size
Language: for any item with text
Type: image, text, interactive resource, moving image
Coverage: GETTY format City, County, State, Country
